# 🍡 말랑 (Mallang)

사진을 AI로 귀여운 말랑이(스퀴시) 캐릭터로 변환하고, 저장하고, 만지고 노는 앱.

> 기획서 v0.1의 MVP 범위(홈 · 사진 업로드 · AI 생성 · 결과 저장 · 보관함)를 웹앱으로 구현한 프로젝트입니다.

## 실행 방법

빌드 과정 없이 정적 파일만으로 동작합니다.

```bash
# 아무 정적 서버로 실행
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000 접속
```

모바일 우선 반응형 UI로, 브라우저에서 바로 사용할 수 있습니다.

## 기능

기획서의 핵심 기능 6가지가 모두 구현되어 있습니다.

| 기능 | 설명 |
|---|---|
| 🏠 홈 | 말랑이 마스코트(꾹 누르면 말랑말랑!)와 시작 버튼 |
| 📷 사진 업로드 | 파일 선택 또는 드래그&드롭, 미리보기 |
| ✨ AI 생성 | 모찌 · 젤리 · 푸딩 · 슬라임 4가지 스타일로 말랑이 변환. 스타일별 시그니처(모찌 귀, 푸딩 카라멜+크림, 젤리 기포, 슬라임 흘러내림) |
| 🖐️ 만지기 | 놀이터에서 스프링 물리 기반 인터랙션 — 꾹 누르면 납작(오래 누르면 펼치기), 끌면 늘어뜨리기, 비비면 주무르기, 휙 튕기면 던지기(벽에 통통). 액션 버튼 5종: 찰랑찰랑·주무르기·납작펴기·빙글빙글·던지기 |
| 🔊 사운드 | WebAudio 합성 효과음 — 탭·뽁·통통·휙·반짝 팡파레·저장 딩·삭제 슉. 홈 화면에서 음소거 토글 |
| 🎨 꾸미기 | 배경 6종 · 모자 6종 · 스티커 8종(최대 12개, 드래그로 위치 이동). 꾸민 상태는 보관함에 함께 저장 |
| 📤 공유 | 이미지(PNG) · GIF(흔들리는 애니메이션, 자체 인코더) · 영상(WebM). Web Share API 지원 기기에서는 바로 공유, 아니면 파일 저장 |
| 🗂️ 보관함 | 저장한 말랑이 목록(꾸민 모습 썸네일) · 상세 보기 · 다운로드 · 삭제 · 다시 꾸미고 놀기 |
| 📱 아이폰 최적화 | 세이프 에어리어(노치·홈 인디케이터) 대응, 바운스 스크롤·더블탭 확대 차단, iOS 영상 내보내기(mp4 폴백), 상태바 색 동기화, PWA(홈 화면에 추가 → 풀스크린 앱 + 오프라인 실행) |

## 프로젝트 구조

```
index.html      # 전체 화면(홈/업로드/생성/결과/놀이터/보관함) 마크업
css/style.css   # 파스텔 톤의 말랑말랑한 UI 스타일
js/store.js     # 보관함 저장소 (localStorage)
js/sound.js     # WebAudio 합성 효과음 + 음소거 토글
js/mallang.js   # 말랑이 생성 엔진 + 꾸미기 합성 (배경/모자/스티커)
js/gif.js       # 자체 GIF89a 인코더 (미디언 컷 팔레트 + LZW)
js/play.js      # 놀이터: 스프링 물리 인터랙션·액션, 꾸미기, 공유/내보내기
js/app.js       # 화면 전환, 업로드, 생성 흐름, 보관함 UI
```

## 웹 배포

`.github/workflows/pages.yml`이 GitHub Pages로 자동 배포합니다.
저장소 **Settings → Pages → Source**를 **GitHub Actions**로 한 번만 설정하면
이후 푸시마다 공개 URL(`https://<owner>.github.io/malang_EO/`)로 배포됩니다.

## 말랑이 생성 엔진

`js/mallang.js`의 `Mallang.generate(imageSrc, style)`가 변환을 담당합니다.

- 현재는 **온디바이스 캔버스 렌더링**으로 동작합니다: 사진을 울퉁불퉁한 블롭 모양으로 자르고, 스타일별 틴트·광택·볼륨감을 입힌 뒤 귀여운 얼굴(눈·볼터치·입)을 그립니다.
- 같은 사진 + 같은 스타일이면 항상 같은 모양이 나오도록 시드 기반으로 렌더링합니다.
- 추후 실제 AI 이미지 생성 API로 업그레이드할 때는 `generate()` 내부 구현만 교체하면 나머지 앱은 그대로 동작합니다.
